# 📊 HR Analytics Dashboard (Excel Project)

## 📌 Project Overview
This project is an interactive **HR Analytics Dashboard** built in Microsoft Excel. It transforms raw employee data into meaningful insights that help HR teams monitor workforce performance, employee attrition, salaries, and demographics.

The dashboard is designed to support data-driven decision-making through KPIs, Pivot Tables, Pivot Charts, and Slicers.

---

## 🎯 Project Objective
The objective of this project is to analyze employee data and provide actionable insights that help organizations improve employee retention, workforce planning, and HR decision-making.

---

## 📈 Key Performance Indicators (KPIs)

- 👥 Total Employees
- ✅ Active Employees
- 🚪 Attrition Count
- 📉 Attrition Rate (%)
- 💰 Average Salary
- 🎂 Average Age
- 💼 Average Experience
- ⚧ Male vs Female Ratio

---

## 📊 Dashboard Features

- Employees by Department
- Attrition by Department
- Gender Distribution
- Salary by Department
- Monthly Joining Trend
- Performance Rating Analysis
- Overtime vs Attrition
- Interactive Slicers

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Functions (COUNTIFS, SUMIFS, AVERAGEIFS, IF, XLOOKUP)

# Business Questions & Answers

## 1. Which department has the highest attrition?

**Answer:**
The **Sales Department** has the highest attrition rate. This indicates that employees in Sales are leaving more frequently than in other departments, suggesting a need to review workload, incentives, and employee engagement.

---

## 3. What is the average salary by department?

**Answer:**
The dashboard shows the average salary for each department. The **IT Department** has the highest average salary, while the **HR Department** has the lowest. This comparison helps management evaluate compensation across departments.

## 4. Which department has the highest number of employees?

**Answer:**
The **Operations Department** has the largest workforce. This suggests it requires the most human resources to support business operations.

**Answer:**
Most employees fall within the **26–35 years** age group. This indicates that the organization has a predominantly young and mid-career workforce.

**Answer:**
Yes. Employees who work overtime have a higher attrition rate compared to those who do not. This suggests that excessive workload may contribute to employee turnover.
